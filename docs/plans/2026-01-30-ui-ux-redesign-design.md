> 日期：2026-01-30
> 状态：已确认，待实施

---

## 一、项目背景

River Hub 是 Riverhub 的个人作品集网站。本次重构目标：

- **视觉升级**：从基础 Bento Grid 升级为大胆前卫的现代风格
- **信息层次优化**：突出核心内容，引导访客注意力
- **品牌调性强化**：体现"技术+设计+AI"三位一体的复合定位

---

## 二、设计定位

### 核心身份
**"用技术实现设计，用 AI 放大创造力"**

三重定位融合：
- 全栈开发者（技术实力）
- 产品设计者（审美眼光）
- AI 探索者（前沿视野）

### 视觉风格
**流体渐变 + 3D 点缀**（参考 Stripe、Vercel）

- 大胆前卫，有视觉冲击力
- 亮色背景，渐变强调
- 重度沉浸式动效体验

## 三、视觉系统

### 配色方案

| 用途 | 色值 | 说明 |
|------|------|------|
| 背景主色 | `#FFFFFF` → `#FAFAFA` | 白到浅灰渐变 |
| 主渐变 | `#8B5CF6` → `#3B82F6` → `#06B6D4` | 紫 → 蓝 → 青 |
| 标题文字 | `#0F172A` | 深灰黑 |
| 正文文字 | `#64748B` | 中性灰 |
| 强调色-紫 | `#8B5CF6` | 产品设计模块 |
| 强调色-蓝 | `#3B82F6` | 前端开发模块 |
| 强调色-青 | `#06B6D4` | AI 应用模块 |

### 字体系统

| 用途 | 字体 | 字重 |
|------|------|------|
| 标题/Slogan | Sora | 600-700 |
| 正文 | Plus Jakarta Sans | 400-500 |
| 代码（如需要） | JetBrains Mono | 400 |

### 视觉元素

- **流体渐变 Blob**：CSS/SVG 动画背景装饰
- **玻璃拟态卡片**：`backdrop-blur` + 半透明边框
- **微妙网格纹理**：增加层次感

## 五、技术架构

### 依赖库

```json
{
  "dependencies": {
    "framer-motion": "^11.x",
    "gsap": "^3.x",
    "@gsap/react": "^2.x",
    "@react-three/fiber": "^8.x",
    "@react-three/drei": "^9.x",
    "lenis": "^1.x"
  }
}
```

| 库 | 用途 |
|---|---|
| framer-motion | React 组件动画、布局动画、手势 |
| gsap + ScrollTrigger | 滚动触发动画、复杂时间线 |
| @react-three/fiber | 3D 渲染（可选，用于 3D 背景） |
| @react-three/drei | Three.js 工具集 |
| lenis | 平滑滚动 |

### 组件结构

```
src/
├── components/
│   ├── ui/
│   │   ├── GradientBlob.tsx      # 流体渐变背景
│   │   ├── GlassCard.tsx         # 玻璃拟态卡片
│   │   ├── TiltCard.tsx          # 3D 倾斜卡片
│   │   ├── AnimatedText.tsx      # 文字入场动画
│   │   ├── ProgressBar.tsx       # 动画进度条
│   │   └── MagneticButton.tsx    # 磁性按钮效果
│   ├── sections/
│   │   ├── Hero.tsx              # 首页 Hero 区
│   │   ├── FeaturedWorks.tsx     # 作品精选区
│   │   ├── SkillMatrix.tsx       # 能力矩阵区
│   │   └── ContactCTA.tsx        # 联系区
│   └── layout/
│       ├── SmoothScroll.tsx      # Lenis 平滑滚动封装
│       └── PageTransition.tsx    # 页面切换动画
└── hooks/
    ├── useScrollProgress.ts      # 滚动进度
    ├── useMousePosition.ts       # 鼠标位置追踪
    └── useTiltEffect.ts          # 3D 倾斜计算
```

### 性能优化

- **动画懒加载**：滚动到视口才初始化
- **GPU 加速**：优先使用 `transform` 和 `opacity`
- **图片优化**：WebP 格式 + 懒加载
- **代码分割**：Three.js 按需加载
